# PsychDataAI Employer Review

PsychDataAI is a behavioral health analytics project that uses Python, SQL, and data visualization to explore mental health disparities across racial and ethnic groups. This version is designed for employer review and highlights the project’s analytical workflow, database structure, and generated visual output.

## Highlights
- Builds a SQLite-backed analytics workflow from Python
- Compares service utilization, access, and treatment against a White baseline
- Presents stereotype-related barriers and their influence on help-seeking and self-advocacy
- Produces a visual comparison chart for quick review

## Files included
- [main.py](main.py) — analysis script
- [psychdataai.db](psychdataai.db) — SQLite database generated by the project
- [../psychdataai_ethnic_comparison.png](../psychdataai_ethnic_comparison.png) — chart output for review
- [review_token.txt](review_token.txt) — project review token
