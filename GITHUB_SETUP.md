# GitHub Portfolio Setup

This project is now initialized as a local Git repository.

## Next steps
1. Create a GitHub repository.
2. Add the remote:
   ```bash
   git remote add origin https://github.com/KayHowse/psychdataai.git
   ```
3. Push the project:
   ```bash
   git branch -M main
   git add .
   git commit -m "Initial portfolio commit"
   git push -u origin main
   ```

## Notes
- Replace the repository URL if you use a different GitHub name or repository name.
- If you want, you can also upload the contents of the github_repo folder as a more polished portfolio landing page.
