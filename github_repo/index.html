PsychDataAI

A behavioral health analytics project using Python, SQL, and data visualization to explore mental health disparities across racial and ethnic groups.

View on GitHub Read the README

Project focus

  • Compares service utilization, access, and treatment across groups
  • Highlights stigma and self-advocacy factors influencing care
  • Shows how Python and SQL can support public-health-style analysis

Skills demonstrated
