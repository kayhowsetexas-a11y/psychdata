# PsychDataAI

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue)](https://www.python.org/)
[![SQLite](https://img.shields.io/badge/Database-SQLite-lightgrey)](https://www.sqlite.org/)
[![Visualization](https://img.shields.io/badge/Charts-Matplotlib-orange)](https://matplotlib.org/)

PsychDataAI is a behavioral health analytics project that uses Python, SQL, and data visualization to explore mental health disparities across racial and ethnic groups. It provides an example of how data can be structured and analyzed to highlight inequities in mental health service utilization, access, treatment, stigma, and self-advocacy.

## Project overview
- Builds a SQLite-backed dataset for public health-style analysis
- Compares selected groups against a White benchmark
- Visualizes service gaps and stigma/self-advocacy scores
- Summarizes the social and cultural factors that may influence help-seeking behavior
